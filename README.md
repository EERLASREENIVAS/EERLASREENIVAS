## 📚 Featured Projects

- [**FlightNest**](https://github.com/ChinthaVamsidharReddy/FlightNest) — Full-stack **flight booking system** using **Angular**, **Spring Boot**, and **MySQL**.  
  Includes flight search, user registration, booking management, and admin dashboard.

- [**Service Hub**](https://github.com/ChinthaVamsidharReddy/Service-Hub) — Scalable full-stack **service booking platform** built with **React**, **Node.js**, and **MySQL**.  
  Enables users to book on-demand services with secure authentication and real-time updates.

- [**Herbal Plant Identifier**](https://github.com/ChinthaVamsidharReddy/Herbal-Plant-Identifier) — ML-powered **Flutter** mobile app using **Python CNN model** for real-time herbal plant identification.  
  Supports multilingual UI and local model inference.

- [**Frontal Face and Eye Pattern Recognition**](https://github.com/ChinthaVamsidharReddy/Frontal-Face-and-Eye-Pattern-Recognition-in-Images) — Image processing project in **Python** using **OpenCV** and **NumPy**.  
  Detects and highlights faces and eyes in images using Haar Cascade classifiers.

- [**Nerchukundam (Educational Website)**](https://github.com/ChinthaVamsidharReddy/Educational-Website) — Web platform developed using **React**, **Node.js**, and **MySQL**.  
  Provides study resources, e-books, placement preparation, and tutorials.
  
- [**ATM Simulation**](https://github.com/ChinthaVamsidharReddy/ATM-Simulation) — Core **Java** project simulating basic ATM operations using **OOP principles** and transaction management.

- [**Portfolio Website**](https://github.com/ChinthaVamsidharReddy/Portfolio) — Personal developer portfolio showcasing projects and skills.

- [**Rice Leaf Disease Prediction**](https://github.com/ChinthaVamsidharReddy) — **Machine Learning project** using **Random Forests** and **CNNs** for disease classification in rice leaves.
